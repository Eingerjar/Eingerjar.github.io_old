<h2>Git branch</h2>
원본 버전은 유지한 상태에서 다른 기능을 추가하고 싶을 때 branch를 사용한다.
branch는 뜻 그대로 가지를 생각하면 이해하기가 싶다.

<h3>사용법</h3>

{% highlight ruby %}
git branch # 뒤에 원하는 branch이름을 작성하면 branch가 현재 HEAD에 생성된다.
